# 1. 常用的Linux命令

```bash
ls -list

mkdir - make directory

pwd  — Print Working Directory

cd — Change Directory

rmdir— Remove Directory # 删除指定的文件夹

rm— Remove # 删除指定文件
	-r 就是向下递归，不管有多少级目录，一并删除
	-f 就是直接强行删除，不作任何提示的意思
cp— Copy # 复制 cp 路径一文件 路径二文件

mv— Move # 移动文件夹或者文件 mv 路径一文件 路径二文件

cat— concatenate and print files # 标准输出

tail — print TAIL(from last) # 输出给定文件最后10行内容,可以使用tail -n N 指定在标准输出上显示文件的最后N行内容。


less -print less # less 按页或按窗口打印文件内容。在查看包含大量文本数据的大文件时是非常有用和高效的。你可以使用Ctrl+F向前翻页，Ctrl+B向后翻页。
```
